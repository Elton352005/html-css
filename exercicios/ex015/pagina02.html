Existe ainda uma melhor oção. Criar um arquivo CSS para agrupar o estilo de todas as páginas do seu site

Ao fazer isso haverá uma sugestão de nome arquivo a ser criado (style.css). Como ele ainda não existe, segunrando "ctrl"+aproximando o mouse do nome, o VSC ja informa que não da pra seguir o link e sugere a criação do arquivo. Basta confirmar e o memso será criado

Dentro deste arquivo CSS criado é que vamos inserir os estilos- referenciando a tag "body" de todo o site

Atenção - É necessário inserir @charset "UTF-8" na primeira linha do arquivo

Definição de estilos

  • Inline
    Estiliza linha a linha(tag) do conteúdo
  • Interno
    Utilizado no "Head" da página e agrupa toda ela
  • Externo
    Criado um arquivo CSS que agrupa todo o conteúdo do site (o estilo formata todas as páginas existentes)
