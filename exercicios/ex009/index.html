A tag "ol" definie o titulo da lista

A tag "li" define os tópicos da lista

Obs. Não há necessidade de enumeração dos tópicos pois as tags ja fazem essa função

A tag "ol" oferce tipos de listas com a tag "type" inserida logo após "ol"

Tais tipos referem-se à Numerico, alfabético minúsculo e maiúsculo, alfabaeto romano minúsculo e maiúsculo etc...

E também é possivel determinar a ordem a qual a lista será apresentada com a tag "start" após a tag "type"

Exemplo:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. escovar os dentes

No exemplo acima determinamos que a lista tem o tipo "numérico" e inicia a contagem de tópicos à aprtir do número 4

Listas não Ordenadas

Para este tipo de lista a tag utilizada é "ul"

Os demais conceitos são extamente os mesmos mencionados anteriormente. Contudo e, neste caso, não haverão numerações ou carcteres que identifiquem as linhas da lista

  • Acordar
  • Ligar para o João
  • Tomar café
  • escovar os dentes
  • Ir para a faculdade
  • Almoçar
  • Ir para o trabalho
  • Voltar para casa
  • Jantar
  • Dormir

A tag "ul" também oferece tipos (Type). Tais tipos referem-se ao caractere utilizado para identificar a linha(a bolinha no incio da linha)

Alguns tipos são "disk", "circle" e "square"
